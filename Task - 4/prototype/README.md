FitStart — Low-fidelity Clickable Prototype

What this is
- A lightweight, local clickable prototype demonstrating a user journey from Welcome → Onboarding → Profile → Goals → Summary.

How to open
- Recommended: open the `prototype` folder in VS Code and run Live Server, or serve the folder with a simple static server.

Quick run (Node.js)

```bash
npx http-server prototype -c-1
# then open http://localhost:8080 in your browser
```

Files
- prototype/index.html — main clickable screens
- prototype/styles.css — low-fi styling
- prototype/app.js — interactions and state
- prototype/annotations.md — annotated design decisions
